How this is calculated
  • API-equivalent value prices real token counts at OpenAI's public GPT-5.5 rates: $5 input · $0.50 cached input · $30 output, per 1M tokens.
  • Effective rate = weekly price ($46.15) ÷ API value at 100% quota. Effective per-token prices are the list prices scaled by that ratio.
  • Monthly figures extrapolate the weekly window using 52/12 weeks per month - a value estimate, not OpenAI's actual cost.
  • Fast vs non-fast mode is not split; source logs only expose aggregate token counters. Source: result/melvynx/codex/2026-05-30T09-13-16Z.
