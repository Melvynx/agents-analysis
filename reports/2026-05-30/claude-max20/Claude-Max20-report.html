How this is calculated
  • API-equivalent value prices real token counts at Claude Opus 4.7 public rates: $5 input · $0.50 cache read · $10 cache write 1h · $25 output, per 1M tokens.
  • The direct figure extrapolates a ~1-day window at 10% quota: factor (1/0.10) × (52/12) ≈ 43.3 to reach monthly at 100% quota. The ×10 step carries real uncertainty.
  • Triangulation: Max 20x = 4× the Max 5x quota and 20× the Pro quota. Scaling those samples gives $21.1k and $21.8k/mo, bracketing the direct $18.9k.
  • Usage is Opus-heavy; a Sonnet/Haiku-heavy month values lower at API rates. API-equivalent value is not Anthropic's real cost. Source: result/taka-foo/claude/data.json.
