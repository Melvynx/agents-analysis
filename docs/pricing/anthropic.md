| Category | Opus Fast price per 1M tokens |

## Notes

- Prices can change. Re-check the official pricing page before publishing a new result.
- API-equivalent subscription analysis compares user value against public API pricing. It does not estimate Anthropic's internal cost.
- Data residency and partner cloud pricing can apply different multipliers.
